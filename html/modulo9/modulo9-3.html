Tema 2 Construcción y uso de las fórmulas para calcular el perímetro y el área del rectángulo.

¿Qué es un rectángulo?

El rectángulo es una figura que se encuentra compuesta por cuatro lados, de los cuales dos tienen una longitud y los dos restantes otras, que además forman cuatro ángulos rectos de 90°. Se puede decir entonces que un rectángulo es un paralelogramo, por el hecho de que los dos pares de lados que lo componen son paralelos respectivamente.

Elementos de un rectángulo.

Los elementos del rectángulo son los siguientes:

  • Vértices:  A, B, C, D.
  • Lados:  AB, BC, DC, AD. Donde AB = DC y AD = BC 
  • Diagonales:  AC, DB.
  • Ángulos interiores:  Todos son rectos (miden 90º).

Perímetro y Área

El perímetro y el área son dos elementos fundamentales en matemáticas.

Perímetro

El perímetro de una figura de dos dimensiones es la distancia alrededor de la figura.

Área

El área de una figura de dos dimensiones describe la cantidad de superficie que cubre la figura.

¿Y el rectángulo?

Perímetro del rectángulo

El perímetro del rectángulo es igual a la  suma de las longitudes de sus  cuatro lados. Se pueden utilizar las siguientes fórmulas para calcular el perímetro:

  • P: Perímetro
  • b: Base
  • h: Altura

Fórmulas:

  • P=b+b+h+h
  • P=2b+2h
  • P=2(b+h)

Área del rectángulo

El área del rectángulo es igual a base por altura. Se puede utilizar la siguiente fórmula para calcular el área:

  • P: Perímetro
  • b: Base
  • h: Altura

Fórmula:

  • A=(b)(h)
