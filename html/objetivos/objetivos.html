Resultados de aprendizaje

Se espera que al finalizar el Curso de matemáticas para niños, aprenderás:

  • Manejar operaciones matemáticas.
  • Usar expresiones aditivas, multiplicativas o mixtas para efectuar cálculos con números naturales.
  • Emplear la multiplicación y división.
  • Conocer sobre el calendario, el reloj, los grados, etc.
  • Aprender sobre los diferentes cuerpos geométricos.
  • Aplicar herramientas matemáticas para resolver problemas de manera autónoma.
  • Comparar diferentes superficies.
  • Reconocer diferentes unidades de medición.

Todo lo anterior te permitirá desarrollar competencias para tu crecimiento profesional y personal.
